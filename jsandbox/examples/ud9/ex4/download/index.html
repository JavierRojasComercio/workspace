Demo: tu primer código JavaScript

En este primer ejemplo puedes ver cómo incluir un script en nuestra web.

Pulsa el botón derecho de tu ratón en esta web y "Ver código fuente de esta página", ahora fíjate cómo la etiqueta "script" contiene el código JavaScript.

Esta primera web muestra el mensaje "Prueba de JavaScript" en un cuadro de alerta cuando carga la página.

Intenta analizar el orden de ejecución del código.

Observa cómo muestra el mensaje antes de "pintar" la página y cómo cuando le das a "aceptar" continúa renderizando el html

Pulsa F5 para recargar la página y comprobar otra vez el orden de ejecución
