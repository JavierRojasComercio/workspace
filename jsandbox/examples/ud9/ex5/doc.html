Demo: tu primer código JavaScript

En este primer ejemplo puedes ver cómo incluir un script embebido en nuestra web.

Fíjate cómo la etiqueta "script" de abajo contiene el código JavaScript.

Esta primera web muestra el mensaje "Prueba de JavaScript" en un cuadro de alerta cuando carga la página.

Intenta analizar el orden de ejecución del código.

Pulsa el botón RUN

Observa cómo muestra el mensaje antes de "pintar" la página

y cómo espera hasta que pulses "Aceptar" para continuar el renderizando el html

De hecho, puedes ver cómo la ejecución de toda la página se para, a la espera de que pulses.

Pulsa RUN para recargar la página y comprobar otra vez el orden de ejecución
