Este ejemplo analizan las principales variables que existen en JavaScript y sus propiedades.

Tipos de datos

Los tipos se dividen en dos grupos, Primitivos y de Objeto.

Tipo Características Elementos definitorios
String Cadenas de texto. Un string es una cadena de caracteres. A cada carácter de una cadena se le asigna una posición, empezando por el primer carácter en la posición 0, el segundo en la posición 1 y así sucesivamente.
Number Valores numéricos. JavaScript solo tiene un tipo de datos numérico. No hay un tipo específico para los números enteros y de coma flotante.
Boolean Representa una entidad lógica Puede tener dos valores: true y false.
null “No valor”. Tiene el valor: null.
undefined Variable sin valor asignado Una variable a la cual no se le haya asignado valor tiene el valor undefined.
Symbol Nuevo en ECMAScript 2015. Symbol es un tipo de datos cuyos valores son únicos e inmutables. Dichos valores pueden ser utilizados como identificadores (claves) de las propiedades de los objetos.
Object Un valor en memoria al que podemos acceder por un identificador. En JavaScript los objetos pueden ser vistos como una collección de propiedades.

Determinación de tipos

El operador typeof devuelve una cadena indicando el el tipo de valor de una variable.

Podemos ver algunos ejemplos:

  let nombre = "Sergio";
  alert(typeof nombre);
  
  let paquetes = 3;
  alert(typeof paquetes); 
  
  let enviado = false;
  alert(typeof enviado);

Conversión de tipos

parseInt() y parseFloat() son funciones creadas para parsear un string y devolver un número si es posible. Los espacios iniciales y finales se ignoran.

JavaScript analiza la cadena para extraer las cifras que encuentre al principio. Estas cifras al principio del string son las que se transforman a tipo numérico. Cuando se encuentra el primer carácter no numerico se ignora el resto de la cadena. Si el primer carácter encontrado no es convertible a número, el resultado será NaN (Not a Number).

Ejemplos: 
        let e1 = parseInt("1Sergio");
        alert(e1);

        let e2 = parseInt("Ser1gio1");
        alert(e2);

        let e2 = parseInt("Sergio1");
        alert(e2);

        let e2 = parseInt("Sergio");
        alert(e2);

Cuando el valor no es un string, JavaScript hace primero una conversión implícita a string. Esta conversión, en el caso de objetos, se hace llamando al método toString(). Podemos reescribir este método para algunos objetos si nos interesa asegurarnos de que devuelvan un string convertible a número.

El dinamismo de Javascript

Los tipos de JavaScript son dinámicos JavaScript tiene tipos dinámicos. Esto significa que la misma variable se puede usar para contener diferentes tipos de datos:

Ejemplos: 
  let variableMareada = "Sergio";
  alert(typeof variableMareada);
  
  variableMareada = 3;
  alert(typeof variableMareada); 
  
  variableMareada = false;
  alert(typeof variableMareada);

La concatenación: el uso de +

Si el + suma o concatena va a depender del contexto. Veamos algunos ejemplos

        let sumaa = 3 + 2;
        let sumab = 3 + "2";
        let sumac = 5 + 2 + "Sergio";
        let sumad = 5 + "Sergio" + 2;
        let sumae = 5 + 2 + "Sergio" + 2 + 4;
        let sumaf = 5 + 2 + "Sergio" + (2 + 4);
Si queremos forzar la concatenación de dos números podemos utilizar las comillas 
        let conc1 = '19' + '67';    // ¿Qué tipo tendrá la variable conc?
        let conc2 = 19 + '67';      // ¿Y ahora?
        let conc3 = '19' + 67;      // ¿Y ahora?
        let conc4 = 19 + 67;      // Finalmente
