Este ejemplo analizan los Arrays y los métodos asociados más importantes

Arrays

Los arrays conjunto de datos ordenados por posiciones y todos asociados en una sola variable. Los datos pueden ser de cualquier tipo de dato, es decir, es posible crear una array que tenga una cadena en la primera posición, un número en el segundo, un objeto en el tercero y así sucesivamente. Podremos acceder a estos distintos datos de manera independiente o agrupándolos. Cabe resaltar que un array es un objeto, por ejemplo:

    let persona = ["Sergio", 22, true];

Existen dos formas de crear un Array y son las siguientes:

    var colores = new Array("rojo", "azul", "verde"); 
    var colores = ["rojo", "azul", "verde"] ;

También tenemos la opción de crear un array vacio

    var colores =  new Array(); 
    var nombres = [];

Hablemos ahora de la forma en que podemos acceder a algunos de los elementos del Array. Esto se logra, a través del nombre establecido junto a dos corchetes, entre ellos la posición del elemento, de la siguiente manera: Tenemos al siguiente array:

      let colores = ["rojo", "azul ", " verde "]; 
      alert(colores[1]);

Métodos más empleados en los Arrays

Convertir un Array en String

Para hacerlo, contamos con dos métodos

      let nombres = ["Pedro", "Pablo", "Luis", "Sergio"];
      let nombresC1 = nombres.toString();
      let nombresC2 = nombres.fruits.join("; ");

Calcular el tamaño de un Array

Para hacerlo contamos con el método length, de la siguiente manera: 

    let nombres = ["Pedro", "Pablo", "Luis", "Sergio"];
    nombres.length;

Añadir elementos

El mmétodo push() agrega un nuevo elemento a una matriz (al final) y devuelve el tamaño del Array:

      let nombres = ["Pedro", "Pablo", "Luis", "Sergio"];
      alert("Ahora el tamaño de 'nombres[]' es de " + nombres.push("Juan"));

El método unshift() agrega un nuevo elemento a una matriz (al principio) y "deshace" los elementos más antiguos. Además, devuelve el tamaño del nuevo Array:

    let nombres = ["Pedro", "Pablo", "Luis", "Sergio"];
    alert("Ahora el tamaño de 'nombres[]' es de " + nombres.unshift("Juan"));

Eliminar elementos

El método pop() elimina el último elemento de un Array:

    let nombres = ["Pedro", "Pablo", "Luis", "Sergio"];
    nombres.pop();

El método shift() elimina el primer elemento de la matriz y "desplaza" todos los demás elementos a un índice más bajo.

    let nombres = ["Pedro", "Pablo", "Luis", "Sergio"];
    nombres.shift();

Dado que las matrices de JavaScript son objetos, los elementos se pueden eliminar mediante el operador de JavaScript delete Sin embargo, en este caso el espacio quedaría vacío:

    let nombres = ["Pedro", "Pablo", "Luis", "Sergio"];
    delete nombres[0];

El uso de delete puede dejar huecos sin definir en la matriz. Utilizar pop() o shift() en su lugar.

Union de Arrays

Para unir Arrays podemos utilizar al función concat() 
    let chicas = ["Ana", "Bea"];
    let chicos = ["Pedro", "Luis"];
    let todos = chicas.concat(chicos);

Podemos unir más de dos arrays añadiendolos dentro del paréntesis. Ejemplo: chicas.concat(chicos, otros)

Ordenar un Array

El método sort() ordena un Array alfabéticamente:

      let nombres = ["Pedro", "Pablo", "Luis", "Sergio"];
      let nombresAux = nombres.sort();

Darle la vuelta a un Array

El método reverse() le da la vuelta al orden de los elementos de un Array:

        let nombres = ["Pedro", "Pablo", "Luis", "Sergio"];
        let nombresAux = nombres.reverse();

Mayor y menor elemento de una matriz

La libreria Math nos da dos métodos maravillosos para este propósito

      let numeros = [2,3,8,5,1];
      let mayor = Math.max(null,numeros);
      let menor = Math.min(null,numeros);
